※ Divergence

1 散度的定义

给定两个定义在同一字母表X上的随机变量X和X^，定义散度(又称为Kullback-Leibler divergence)：

(1-1)DKL(X||X^)=DKL(PX||PX^):=EX[log2PX(X)PX^(X)]=∑x∈XPX(x)log2PX(x)PX^(x)

换句话说，散度就是分布PX相比PX^的对数似然比值的期望。说人话可以这样理解：

It is the amount of information lost when Q is used to approximate P

It is the expected number of extra bits required to code samples from P using a code optimized for Q rather than the code optimized for P

2 散度的动机

如果使用一个错误的分布来近似真实分布，那么事实上每一个元事件需要多额外的信息量的期望是多少？

很显然，期望是概率加权的均值，这个概率是事件真实概率，而额外的信息量怎么表示？可以用自信息的差，也就是：

−log2PX^(x)−(−log2PX(x))=log2PX(x)PX^(x)

这就类似于计算一个编码系统，当它采用有偏差的分布来设计最优编码方案时，它在实际应用中实际每个编码字母平均付出的额外代价。

2.1 交叉熵

注意到散度公式可以写为如下形式：

DKL(X||X^)=∑x∈XPX(x)·log21PX^(x)−∑x∈X−PX(x)·log2PX(x)(2-1)=CH(X,X^)−H(X)

其中，我们将CH(·)定义为交叉熵(Cross Entrophy)，表示使用一个估计分布PX^来设计编码方案时候，平均需要的信息量。很显然交叉熵非负。当估计分布等于真实分布时候，交叉熵就是熵。即CH(X,X)=H(X)。

3 散度的属性

3.1 非负性 Nonnegativity of divergence

(3-1)DKL(X||X^)≥0

等式当且仅当对于∀x∈X,PX(x)=PX^(x)。

Proof：

DKL(X||X^)=DKL(PX||PX^)=∑x∈XPX(x)log2PX(x)PX^(x)≥(∑x∈XPX(x))log2∑a∈XPX(a)∑b∈XPX^(b)=0

根据log-sum不等式，上述不等式成立，且等号当且仅当对于∀x∈X,PX(x)=PX^(x)。

3.2 互信息与散度的关联 Mutual inforamtion and divergence

(3-2)I(X;Y)=D(PX,Y||PX×PY)

根据定义容易得证。

3.3 Refinement cannot decrease divergence

设PX和PX^分别是PU和PU^的细化，那么：

(3-3)DKL(PX||PX^)≥DKL(PU||PU^)

3.4 散度的缺陷

使用散度作为衡量两个分步之间的测度有一个缺陷，就是散度不具有对称性，同时它也不满足三角不等式，所以更不能作为真正的距离(distance)或指标(metric)。

什么是真正的距离或者指标呢？它是一个满足一下条件的函数。

给定一个非空集合A，函数d:A×A→[0,∞) 如果满足以下三个性质，则可以称之为距离或者指标。

  1. 非负性：∀a,b∈A,d(a,b)≥0成立，并且等号当且仅当a=b时成立。
  2. 对称性：∀a,b∈A,d(a,b)=d(b,a)。
  3. 三角不等性：∀a,b,c∈A,d(a,b)+d(b,c)≥d(a,c)。

 

※ Variational distance (L1-distance)

1 变分距离的定义

散度不是真正的距离，由此引出变分距离。对于具有相同字母表X的两个分布PX和PX^，变分距离定义为：

||PX−PX^||:=∑x∈X|PX(x)−PX^(x)|

2 变分距离的引论

||PX−PX^||=2·supE⊂X|PX(E)−PX^(E)|=2·∑x∈X;PX(x)>PX^(x)(PX(x)−PX^(x))

其中符号sup表示集合的最小上界。

Proof:

  1. 先证第二个等号右侧等于第一个等号左侧。

设集合A:=x∈X:PX(x)>PX^(x)，且Ac为其补集，显然有：

1=PX(A)+PX(Ac)=PX^(A)+PX^(Ac)⇒PX^(Ac)−PX(Ac)=PX(A)−PX^(A)

所以，

||PX−PX^||=∑x∈X|PX(x)−PX^(x)|=∑x∈A|PX(x)−PX^(x)|+∑x∈Ac|PX(x)−PX^(x)|=∑x∈A(PX(x)−PX^(x))+∑x∈Ac(PX^(x)−PX(x))=∑x∈A(PX(x)−PX^(x))+PX^(Ac)−PX(Ac)=∑x∈A(PX(x)−PX^(x))+PX(A)−PX^(A)=∑x∈A(PX(x)−PX^(x))+∑x∈A(PX(x)−PX^(x))=2·∑x∈A(PX(x)−PX^(x))
  1. 再证明第一个等号两侧相等

对于任意集合E⊂X，根据绝对值不等式易得：

||PX−PX^||=∑x∈X|PX(x)−PX^(x)|=∑x∈E|PX(x)−PX^(x)|+∑x∈Ec|PX(x)−PX^(x)|≥|∑x∈E[PX(x)−PX^(x)]|+|∑x∈Ec[PX(x)−PX^(x)]|=|PX(E)−PX^(E)|+|PX(Ec)−PX^(Ec)|=|PX(E)−PX^(E)|+|PX(E)−PX^(E)|=2·|PX(E)−PX^(E)|

所以，||PX−PX^||≥2·supE⊂X|PX(E)−PX^(E)|。

根据最小上界定义易得

2·supE⊂X|PX(E)−PX^(E)|≥2·|PX(A)−PX^(A)|=|PX(A)−PX^(A)|+|PX^(Ac)−PX(Ac)|=|∑x∈A[PX(x)−PX^(x)]|+|∑x∈Ac[PX(x)−PX^(x)]|=∑x∈A|PX(x)−PX^(x)|+∑x∈Ac|PX(x)−PX^(x)|=||PX−PX^||

所以第一个等号只能相等得证。

综合1、2推理得证。
